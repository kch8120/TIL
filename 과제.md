<h2>스캐너</h2>
<h3>지금까지 우리는 java에서 변수에 숫자를 대입했다.<br>
대입하기만 했다.<br><br>
우리가 직접 입력하지는 않았다.<br><br>
하지만 이제는 아니다.<br><br>
우리도 입력을 할 수 있다!<br><br>
스캐너(scanner)를 쓰면 가능하다!<br><br>
<br>
하지만 우리는 java에서 바로 스캐너를 쓸 수 없다.<br><br>
스캐너를 쓰려면 절차가 필요하기 때문이다...<br><br>
스캐너를 쓰려면 java에게 스캐너를 쓰겠다고 알려야 한다.<br><br>
그게 바로<br>Scanner scanner = new scanner(System.in);<br><br>
구문이다.<br></h3><br>
<h4>(c언어에서 우리가 include stdio.h를 쓰는 것과 같다!)</h4><br><br>
<h3>만약 우리가 num이라는 변수에 정수를 입력받고 싶다고 하자.<br><br>
정수를 입력하려면 우선<br>Scanner scanner = new scanner(System.in);<br>을 쓰고,<br><br>
int(정수이므로) num = scanner.nextInt();<br>를 쓰면 정수를 입력할 수 있다.<br><br>
이번에는 str이라는 변수에 문자열을 입력하고 싶다고 하자.<br><br>
그렇다고 코드를 String str = scanner.nextStr();<br><br>
이렇게 쓰면 안된다!!<br><br>
문자열을 입력하는 코드는 str = scanner.next"Line"();이다!!!<br><br>
그렇다면 문자열을 계속 입력 받다가 "exit"라는 문장이 등장하면 프로그램을 종료하게 만들어보자.<br><br>
코드를 작성하면<br><br>
Scanner scanner = new scanner(System.in);<br><br>
while(true) {<br>
&nbsp;&nbsp;&nbsp;&nbsp;System.out.print("문자열을 입력하세요(exit:종료):");<br>
&nbsp;&nbsp;&nbsp;&nbsp;String str = scanner.nextLine();<br>
&nbsp;&nbsp;&nbsp;&nbsp;if (str.equals("exit")) {<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;System.out.println("프로그램을 종료합니다.");<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;break;<br>
}
System.out.println("입력한 문자열: " + str);<br><br>
이렇게 된다.<br><br>
지금까지 스캐너에 대해서 알아봤습니다.

<h2>배열</h2><br><br>
<h3>배열은 같은 타입의 변수를 사용하기 편하게 하나로 묶어둔 집합이다.<br><br>
만약 배열이 없다면 우리는 변수를 선언하고 대입하고 .... 변수를 선언하고 대입하고.......(무한반복) 해야한다.<br><br>
반복문을 쓰면 되지 않냐고?<br><br>
과연 그럴까?<br><br>
예를 들어보자.<br><br>
학생 5명이 있다.<br><br>
학생들의 점수를 출력하는 프로그램을 만들어보자.<br><br><br>

int student2 = 80;<br><br>
int student3 = 70;<br><br>
int student4 = 60;<br><br>
int student5 = 50;<br><br></h3>

그럼 어떻게 해야 할가?<br><br>
그렇다, 바로 배열을 쓰면 된다!<br><br>
그런데 위의 코드를 배열로 바꾸려면 절차가 필요하다.<br><br><br>

우선 배열을 선언해보자.<br><br>
int[] students;<br><br>
이렇게 쓰면 정수를 넣을 수 있는 배열 "students"가 선언된다.<br><br>

배열을 생성해보자.<br><br><br>
students = new int[5];<br><br>
이렇게 쓰면 총 5개의 int형 변수"students"가 생성된다.<br><br></h3>
<h4>(그런데 배열은 사실 이름이 없는 존재다.<br><br>
배열을 생성하고 나면 메모리 어딘가에 이 배열에 접근할 수 있는 주소를 만드는데,<br><br>
이를 참조값이라 한다.<br><br>
int[] students 변수는 new int[5]라는 배열의 참조값을 가지고 있는 것일 뿐이다.<br><br>
배열에 접근할 수 있는 녀석이 students이기 때문에,<br><br>
students에 작업을 하는 것이다.)</h4><br><br>
<h3>

왜냐면 현재 students는 [, , , ,]인 상태다.<br><br>

<h2>2차원 배열</h2><br><br>
<h3>2차원 배열은 말 그대로 형태가 2차원 모양인 배열을 말한다.<br><br>
여기서 1차원은 선, 2차원은 면이다.<br><br>
<b>[,][,][,][,][,]<br></b>
1차원<br><br>
<b>[ [,],[,],[,],<br>
&nbsp;&nbsp;[,],[,],[,],
&nbsp;&nbsp;[,],[,],[,] ]<br></b>
2차원<br><br><br>
<img src="https://search.pstatic.net/common/?src=https%3A%2F%2Fdbscthumb-phinf.pstatic.net%2F3523_000_1%2F20141029030721786_68CGOV7TQ.jpg%2Fka7_108_i2.jpg%3Ftype%3Dm4500_4500_fst_n&amp;type=sc960_832" width ="500" height ="300">
위의 2차원 배열 사진을 보자.<br><br>
세로가 행(row)이고 가로가 열(column)이다.<br><br>
위 사진 처럼  arr의 인덱스(위치)는<br><br>
arr[행][열]이다.(이제 0부터 시작하는 이유는 알아야 한다.)<br><br>
위 사진에는 안나왔지만 배열에 값을 대입한다면<br><br><br>
arr[0][0] = 1;<br><br>
arr[0][1] = 2;<br><br>
arr[0][2] = 3;<br><br>
arr[1][0] = 4;<br><br>
arr[1][1] = 5;<br><br>
arr[1][2] = 6;<br><br>
이렇게 대입할 수 있다.<br><br>
출력하려면<br><br><br>
System.out.print(arr[0][0] + " ");<br><br>
System.out.print(arr[0][1] + " ");<br><br>
System.out.print(arr[0][2] + " ");<br><br>
System.out.println();<br><br>
<br><br>
System.out.print(arr[0][0] + " ");<br><br>
System.out.print(arr[0][1] + " ");<br><br>
System.out.print(arr[0][2] + " ");<br><br>
System.out.println();<br><br>
이렇게 작성하면 된다.<br><br>
이 코드도 긴 것같은 느낌이 들지 않는가?<br><br>
이제 이 코드를 리펙토링 해보자.<br><br>

이 코드를 리팩토링 해보면<br><br>
int[][] arr = new int[2][3];<br><br>
arr[0][0] = 1;<br><br>
arr[0][1] = 2;<br><br>
arr[0][2] = 3;<br><br>
arr[1][0] = 4;<br><br>
arr[1][1] = 5;<br><br>
arr[1][2] = 6;<br><br>
for (int row = 0; row < 2; row++) {<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;for (int column = 0; column <3; column++) {<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;System.out.print(arr[row][column] + " ");<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;}<br><br>
System.out.println();<br><br>
}<br><br>
이렇게 작성 된다.<br><br>
하지만, 일반 배열(1차원 배열)에서는 배열의 선언과 생성과 대입이 한 번에 가능했다.<br><br>
2차원 배열에서도 가능할까?<br><br>
결론적으로, 가능하다.<br><br>
코드를 작성해보면<br><br>
int[][] arr = {<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;{1,2,3},<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;{4,5,6}<br>
};<br><br>
for (int row = 0; row < 2; row++) {<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;for (int column = 0; column <3; column++) {<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;System.out.print(arr[row][column] + " ");<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;}<br><br>
System.out.println();<br><br><br>

이렇게 된다.<br><br>
코드가 정말 간단해졌다^O^<br><br>
그렇다면 이제 1~9까지의 숫자를 3씩 끊어서 2차원배열로 만들어보자.<br><br>
코드는<br><br><br>
int[][] arr = new int[3][3];<br><br>
